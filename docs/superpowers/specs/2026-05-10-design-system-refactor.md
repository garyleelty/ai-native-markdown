> 日期：2026-05-10

## 一、设计哲学

**"工具的美感"**

不追求装饰性的华丽，而追求功能性的优雅。每一个视觉元素都服务于效率，每一处细节都体现专业。

### 参考标杆

- **Obsidian**：深色模式的层次感、侧边栏的信息密度
- **VS Code**：工具按钮的精确反馈、状态栏的信息丰富度
- **Zed**：极简但精致的界面元素、流畅的动画
- **Cursor**：AI 功能的视觉融合、现代感与专业感的平衡

## 三、字体系统

| 用途 | 字体栈 | 大小 | 字重 | 行高 |
|------|--------|------|------|------|
| **界面标题** | Inter, -apple-system | 13px | 600 | 1.4 |
| **界面正文** | Inter, -apple-system | 12px | 400 | 1.5 |
| **编辑器** | JetBrains Mono, Fira Code | 14px | 400 | 1.75 |
| **预览正文** | -apple-system, Inter | 15px | 400 | 1.8 |
| **预览标题** | -apple-system, Inter | 按层级 1.5em-2em | 600-700 | 1.3 |
| **代码块** | JetBrains Mono | 13px | 400 | 1.65 |
| **状态栏** | Inter | 11px | 500 | 1.4 |

### 字体特性

- 启用 `font-feature-settings: 'tnum'` 让数字等宽（适合行号、状态栏数字）
- 中文使用系统默认（PingFang SC / Microsoft YaHei），不强制指定

## 五、圆角系统

| Token | 值 | 用途 |
|-------|-----|------|
| `--radius-sm` | 4px | 小按钮、标签、输入框 |
| `--radius-md` | 6px | 普通按钮、卡片 |
| `--radius-lg` | 8px | 大卡片、面板 |
| `--radius-xl` | 12px | 浮层、模态框 |

**原则**：精致专业风使用较小的圆角，保持界面的锐利感和工具感。

## 六、阴影与边框

| 效果 | 值 | 用途 |
|------|-----|------|
| `--border-subtle` | `1px solid rgba(255,255,255,0.06)` | 深色主题分隔线 |
| `--border-light` | `1px solid rgba(0,0,0,0.08)` | 浅色主题分隔线 |
| `--shadow-sm` | `0 1px 2px rgba(0,0,0,0.3)` | 轻微阴影 |
| `--shadow-md` | `0 4px 12px rgba(0,0,0,0.4)` | 下拉菜单、浮层 |
| `--shadow-lg` | `0 8px 24px rgba(0,0,0,0.5)` | 模态框、大浮层 |
| `--glow-accent` | `0 0 12px rgba(124, 158, 248, 0.3)` | 强调发光 |

**原则**：深色主题使用微妙的边框而非阴影来区分层级，阴影仅在浮层上使用。

## 七、动画系统

| 场景 | 时长 | 缓动函数 |
|------|------|----------|
| 按钮悬停 | 150ms | `ease-out` |
| 面板展开/折叠 | 250ms | `cubic-bezier(0.4, 0, 0.2, 1)` |
| 内容切换 | 200ms | `ease-in-out` |
| 拖拽调整 | 实时 | 无过渡 |
| 加载/骨架屏 | 800ms | `ease-in-out`（循环） |

## 八、组件规范

### 8.1 按钮

#### 工具栏按钮（Toolbar Button）

- 尺寸：28px x 28px
- 默认：透明背景，`--text-muted` 颜色
- 悬停：`--bg-hover` 背景，`--text-secondary` 颜色
- 激活：`--bg-active` 背景，`--accent` 颜色
- 选中：`--accent-soft` 背景，`--accent` 颜色，1px `--accent` 边框

#### 主要按钮（Primary Button）

- 高度：32px
- 背景：`--accent` -> `--accent-hover` 渐变
- 文字：白色，13px，600 字重
- 圆角：`--radius-md`
- 悬停：亮度提升，轻微上移 `translateY(-1px)`

### 8.2 输入框

#### 文本输入

- 高度：32px
- 背景：`--bg-surface`
- 边框：1px `--border-subtle`
- 圆角：`--radius-sm`
- 聚焦：边框变为 `--accent`，添加 `--glow-accent` 阴影

### 8.3 标签页（Tabs）

- 高度：36px
- 未选中：透明背景，`--text-secondary`
- 选中：`--bg-surface` 背景，`--text-primary`，底部 2px `--accent` 指示线
- 悬停：`--bg-hover` 背景

### 8.4 文件树项

- 高度：28px
- 内边距：左 8px，右 8px
- 悬停：`--bg-hover` 背景，左侧出现 2px `--accent` 指示线
- 选中：`--accent-soft` 背景，`--accent` 颜色，左侧 2px `--accent` 指示线
